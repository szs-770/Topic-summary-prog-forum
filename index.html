הגעתם לאשכול מקצועי עמוס בתגובות ודיונים ארוכים? הסקריפט הזה מסכם עבורכם כל דיון בפורום פרוג לשורות התחתונות והמסקנות המעשיות בלחיצת כפתור אחת.

התקן את הסקריפט

יכולות הסקריפט

תמציות ממוקדות מקצועית

קורא את עמודי הדיון מאחורי הקלעים ומפיק סיכום נקי של הנקודות המרכזיות, הפתרונות הטכניים והמסקנות, מבלי לאבד את ההקשר החשוב.

דו-שיח חקירתי עם הדיון

צריכים להעמיק בפרט מסוים? תיבת הצ'אט המובנית בתחתית החלונית מאפשרת לכם לשאול את ה-AI שאלות ישירות על בסיס הטקסט המקורי שנכתב באשכול.

ארכיון גרסאות מובנה

הסקריפט שומר את 20 הסיכומים האחרונים שביצעתם לטעינה מיידית, ומאפשר לדפדף בקלות לאחור בין גרסאות סיכום קודמות של אותו נושא.

ערכת נושא מותאמת

תמיכה מלאה ודינמית במצב לילה בהתאם להגדרות הפורום והמערכת שלכם, לחוויית קריאה יוקרתית ונוחה בכל שעות היממה.

מדריך התקנה מהיר

1

התקנת מנהל הסקריפטים

ודאו שמותקן בדפדפן שלכם התוסף Tampermonkey. לאחר מכן, לחצו על כפתור ההתקנה המוזהב בראש עמוד זה ואשרו את הפעלתו.

2

הפעלת מצב מפתח (חובה!)

כדי לאפשר לדפדפן להריץ את הסקריפט, יש להיכנס להגדרות התוספים בדפדפן (בכרום: `chrome://extensions/`) ולהפעיל את המתג "מצב מפתח" (Developer mode) בראש המסך.

3

הוצאת מפתח API חינמי

היכנסו ל-Google AI Studio, לחצו על Get API key, ייצרו מפתח אישי והעתיקו אותו. בהפעלה הראשונה של הסקריפט בפרוג, הדביקו אותו בחלונית והמערכת תתחיל לעבוד.
