# Topic-summary-prog-forum

# Prog Forum Topic Summarizer - Professional Edition 🚀

סקריפט AI מתקדם המותאם במיוחד עבור **פורום פרוג**, המאפשר לסכם אשכולות ודיונים ארוכים ומורכבים לשורות תחתונות ומסקנות מעשיות בלחיצת כפתור אחת, באמצעות מודל Gemini המתקדם של גוגל.

🔗 **[לכניסה לאתר הרשמי והצגת הדגמות חיות](https://szs-770.github.io/Topic-summary-prog-forum/)**

---

## ✨ תכונות מרכזיות (Key Features)

* **סיכום אשכולות מלא:** שליפת כל עמודי הדיון מאחורי הקלעים והפקת תמצית ממוקדת ומסודרת בנקודות, ללא הגבלת אורך.
* **צ'אט מובנה מבוסס הקשר (Chat):** אפשרות לשאול את ה-AI שאלות והבהרות ישירות על גבי תוכן האשכול מתוך חלונית הסיכום.
* **ניהול גרסאות והיסטוריה:** שמירה מקומית של 20 הסיכומים האחרונים ואפשרות דפדוף בין גרסאות סיכום שונות של אותו אשכול.
* **התאמה מלאה למצב לילה (Dark Mode):** התממשקות אוטומטית וחלקה לעיצוב ולערכת הנושא של הפורום.
* **אינטגרציה עם עורך ההודעות:** כפתור "פרסם" המאפשר לשתול את הסיכום ישירות בתיבת התגובה של הפורום בקליק.

---

## 🛠️ הוראות התקנה מהירות (Installation)

1. ודאו שמותקן בדפדפן שלכם מנהל הסקריפטים **Tampermonkey**.
2. **חובה:** הפעילו את **"מצב מפתח" (Developer mode)** בדף הגדרות התוספים של הדפדפן שלכם (`chrome://extensions/`).
3. לחצו על הקישור הבא להתקנת הסקריפט:
   👉 **[לחצו כאן להתקנת הסקריפט (prog-ai.user.js)](https://github.com/szs-770/Topic-summary-prog-forum/raw/refs/heads/main/prog-ai.user.js)**
4. הנפיקו מפתח API חינמי ב-[Google AI Studio](https://aistudio.google.com/) והזינו אותו בהפעלה הראשונה של הסקריפט בפורום.
